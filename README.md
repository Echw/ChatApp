# Chat App


## How to run it locally?

Clone the project

```bash
  git clonehttps://github.com/Echw/ChatApp.git
```

Go to the project directory

```bash
  cd ChatApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
    
## Tech Stack

**Client:** React, TypeScript, Styled-components
